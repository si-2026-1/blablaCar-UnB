# Requisitos

Este documento organiza os requisitos principais do MVP de forma simples e direta.

## Requisitos funcionais

- Cadastro e login institucional.
- Criacao de perfil.
- Cadastro basico de veiculo para motorista.
- Publicacao de carona.
- Busca de carona.
- Solicitacao e aceite de vaga.
- Chat temporario apos aceite.
- Sugestao de custo compartilhado.
- Encerramento da viagem.
- Avaliacao.
- Denuncia ou bloqueio basico.

## Requisitos nao funcionais

- Privacidade de localizacao e contato.
- Uso confortavel em celular.
- Desempenho adequado em busca e publicacao.
- Protecao dos dados.
- Disponibilidade durante horarios relevantes para estudantes.

## Observacoes de escopo

- O sistema e de caronas solidarias e nao tem finalidade lucrativa.
- A plataforma pode sugerir rateio, mas nao processa pagamentos.
